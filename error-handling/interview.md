# Error Handling Interview Questions

1. **What is the purpose of the finally block?**
   - To execute code regardless of whether an error was thrown or caught (e.g., hiding a loading spinner).

2. **How do you handle errors in Promises?**
   - Using the `.catch()` method or the second argument of `.then()`.

3. **What is the difference between Throwing and Catching an error?**
   - Throwing (`throw`) signals that an exceptional condition has occurred. Catching (`catch`) is the process of responding to that signal.
