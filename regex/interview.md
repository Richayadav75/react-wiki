# Regex Interview Questions

1. **Difference between .test() and .exec()?**
   - `test()` returns a boolean. `exec()` returns an array of information about the match (or `null`).

2. **What are the common flags in regex?**
   - `g` (global), `i` (case-insensitive), `m` (multi-line).

3. **What does the `^ ` and `$ ` anchors represent?**
   - `^ ` matches the start of the string. `$ ` matches the end of the string.
