# JSON Interview Questions

1. **Difference between JSON.stringify() and JSON.parse()?**
   - `stringify` converts a JS object to a JSON string. `parse` converts a JSON string back into a JS object.

2. **What data types are NOT supported by JSON?**
   - Functions, `undefined`, and Symbols. They are ignored or converted to `null` during stringification.

3. **Can JSON have comments?**
   - No, the official JSON standard does not support comments.
