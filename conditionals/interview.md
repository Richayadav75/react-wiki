# Conditionals Interview Questions

1. **When would you use a `switch` statement over `if-else`?**
   - Use `switch` when you have multiple discrete values to check against a single variable, as it can be more readable and slightly more performant for many cases.

2. **What is a "Ternary Operator"?**
   - A shorthand for an `if-else` block: `condition ? exprIfTrue : exprIfFalse`.

3. **What are "Falsy" values in JavaScript?**
   - `false`, `0`, `-0`, `0n` (BigInt zero), `""` (empty string), `null`, `undefined`, and `NaN`.
