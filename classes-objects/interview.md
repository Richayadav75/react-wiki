# Classes & Objects Interview Questions

1. **What is the constructor() method?**
   - A special method for creating and initializing an object instance created with a class.

2. **Difference between Static and Instance methods?**
   - Static methods are called on the class itself (e.g., `Math.random()`). Instance methods are called on individual objects created from the class.

3. **How does inheritance work in classes?**
   - Using the `extends` keyword and the `super()` call to invoke the parent class constructor.
