# Data Types Interview Questions

1. **What are the 7 primitive data types in JavaScript?**
   - String, Number, BigInt, Boolean, Undefined, Null, and Symbol.

2. **What is the difference between `null` and `undefined`?**
   - `undefined` means a variable has been declared but not assigned a value.
   - `null` is an assignment value that represents the intentional absence of any object value.

3. **Why is `typeof null` returning "object"?**
   - This is a known bug in JavaScript that persists for backward compatibility.
